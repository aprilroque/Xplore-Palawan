# Xplore-Palawan

# Description

A mobile travel guide application developed using React Native that promotes sustainable tourism in Palawan, Philippines. 
The application provides travelers with information about tourist destinations, local foods, restaurants, eco-tourism practices, 
and travel guides while supporting local communities and sustainable economic growth.

# Features

* Tourist destination guide
* Local food and restaurant recommendations
* Interactive maps and location services
* Offline access to travel information
* Favorites and category filtering
* Eco-tourism tips and travel guides
* Dark mode support
* Data synchronization
* User-friendly interface
